### **網站功能**
#### 🗺️ **互動地圖**
- 滑鼠滾輪：縮放地圖
- 拖曳：移動地圖
- 點擊圓點：查看測站詳細資訊
- 懸停：快速顯示測站名稱和 AQI

#### 📊 **統計面板**
- **總測站數**: 全台監測站總數
- **平均 AQI**: 所有測站平均空氣品質指數
- **等級分布**: 良好/普通/不健康測站數量

#### 🔄 **數據更新**
- **自動更新**: 每 5 分鐘自動重新整理
- **手動更新**: 點擊 "🔄 重新整理數據" 按鈕
- **即時性**: 數據來自環境部 API

#### 📥 **資料匯出**
- 點擊 "📥 匯出 CSV" 按鈕
- 自動下載完整的測站資料
- 包含所有欄位：測站名稱、AQI、距離等

## 🎨 AQI 顏色說明

| 顏色 | AQI 範圍 | 等級 | 意義 |
|------|----------|------|------|
| 🟢 綠色 | 0-50 | 良好 | 空氣品質令人滿意 |
| 🟡 黃色 | 51-100 | 普通 | 空氣品質可接受 |
| 🔴 紅色 | 101+ | 不健康 | 敏感族群應減少戶外活動 |

## 🔧 故障排除

### **網站無法開啟**
1. 檢查網路連線
2. 嘗試其他瀏覽器
3. 清除瀏覽器快取
4. 確認網址正確: `https://hankc0215.github.io/HW1_AQI_Monitor/`

### **地圖無法載入**
1. 等待幾秒鐘讓地圖載入
2. 重新整理頁面
3. 檢查網路連線

### **數據無法更新**
1. 點擊 "🔄 重新整理數據" 按鈕
2. 檢查環境部 API 狀態
3. 稍後再試
