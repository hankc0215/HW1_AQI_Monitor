專案概述

本專案分析台灣避難收容處所與空氣品質監測站的關聯性，評估各避難所在空氣污染事件中的風險等級，為公共安全決策提供科學依據。

資料清理

從 5,973 筆原始資料清理至 5,925 筆高品質資料，移除 21 筆位置錯誤和海上異常。

空間分析

使用 Haversine 公式計算避難所與最近 AQI 測站的距離，建立空間關聯模型。

風險評估

基於 AQI 值和設施類型，將避難所分類為 High Risk、Warning、Safe 三個等級。

分析方法

最近鄰居演算法

使用 Haversine 公式計算兩點間距離，為每個避難所找出最近的 AQI 監測站。

風險分類邏輯

High Risk: AQI > 100 | Warning: AQI > 50 + 室外 | Safe: 其他情況

情境模擬設計

將高雄美濃測站 AQI 強制設為 150，驗證風險標籤邏輯正常運作。

重要發現

資料品質問題

發現 18 筆明顯錯誤的避難所位置和 3 筆海上異常，已全部清理。

屬性缺失挑戰

原始資料完全缺乏室內/室外資訊，透過關鍵字分析建立分類邏輯。

技術挑戰

API 限制、效能優化、邊界定義等技術問題，均已找到解決方案。
