## 🎯 專案概述

本專案分析台灣避難收容處所與空氣品質監測站的關聯性，評估各避難所在空氣污染事件中的風險等級，為公共安全決策提供科學依據。

### ✨ 核心成果

- 📊 **資料清理**: 從 5,973 筆原始資料清理至 5,925 筆高品質資料
- 🎯 **風險分析**: 識別 935 個 High Risk 和 378 個 Warning 避難所
- �️ **空間分析**: 使用 Haversine 公式計算最近測站距離
- � **完整文檔**: 審計報告、反思報告、使用說明

## 🛠️ 技術規格

### **分析技術**
- **Python 3.7+**: 主要開發語言
- **Pandas**: 資料處理與分析
- **Requests**: HTTP 請求處理
- **Folium**: 互動式地圖生成
- **Python-dotenv**: 環境變數管理

### **數據處理**
- **環境部 API**: 即時 AQI 數據來源
- **Haversine 公式**: 地理距離計算
- **JSON**: 數據交換格式
- **CSV**: 資料匯出格式

##  分析結果

### **風險分布統計**
| 風險等級 | 數量 | 百分比 | 條件 |
|----------|------|--------|------|
| **High Risk** | 935 | 15.8% | AQI > 100 |
| **Warning** | 378 | 6.4% | AQI > 50 + 室外 |
| **Safe** | 4,612 | 77.8% | 其他 |
| **總計** | 5,925 | 100% | - |

### **資料清理歷程**
- **原始資料**: 5,973 筆避難收容處所
- **第一次清理**: 移除 18 個問題項目 → 5,928 筆
- **最終清理**: 移除 3 個海上異常 → 5,925 筆，但仍有多筆位於海上的資料，需靠人工清除

---

## 🎨 重要發現

### **資料品質問題**
- **位置錯誤**: 18 筆明顯錯誤的避難所位置
- **海上異常**: 3 筆座標位於台灣海峽
- **屬性缺失**: 室內/室外資訊完全缺失

### **技術挑戰**
- **API 限制**: 逆地理編碼請求頻率限制
- **效能優化**: 大量地圖標記的渲染問題
- **邊界定義**: 台灣各縣市邊界缺乏官方數位資料
