# AQI 避難所風險分析審計報告

## 專案概述
本專案分析台灣避難收容處所與空氣品質監測站的關聯性，評估各避難所在空氣污染事件中的風險等級。

## 資料處理歷程

### 原始資料
- **來源**: data.gov.tw 資料集 73242 - 避難收容處所點位檔
- **原始筆數**: 5,973 筆避難收容處所記錄
- **座標系統**: EPSG:4326 (WGS84 經緯度座標)

### 資料清理過程

#### 第一階段：問題識別與移除
1. **位置錯誤移除** (18 筆)
   - 忠孝國小：記錄彰化市但座標不符
   - 松林社區活動中心：新竹縣新豐鄉位置錯誤
   - 霧峰區重複設施：移除重複的國中、國小地下室
   - 新豐社區活動中心：記錄南投縣但應為新竹縣
   - 土牛活動中心、高雄市大樹區姑山國小、湖口鄉信勢國小禮堂等

2. **室內/室外屬性補充**
   - 分析避難所名稱關鍵字（地下室、禮堂、戶外、廣場等）
   - 根據設施類型推斷（國小、國中、活動中心等）
   - 分類結果：4,895 個室內，1,033 個室外，16 個未知

#### 第二階段：海上異常清理
3. **海上異常移除** (3 筆)
   - 臺中市東區樂業國小：座標位於台灣海峽
   - 峰谷國小南棟教室：座標位於台灣海峽
   - 追分國小多功能教室：座標位於台灣海峽

### 最終資料集
- **清理後筆數**: 5,925 筆避難收容處所
- **資料完整性**: 所有避難所座標均在台灣陸地範圍內
- **屬性完整性**: 室內/室外分類完整

## 分析方法

### 空間分析方法
1. **最近鄰居演算法**
   - 使用 Haversine 公式計算兩點間距離
   - 為每個避難所找出最近的 AQI 監測站
   - 計算公式：`distance = 2 * R * arcsin(√(sin²(Δφ/2) + cos φ1 ⋅ cos φ2 ⋅ sin²(Δλ/2))`

2. **風險分類邏輯**
   - **High Risk**: 最近 AQI > 100
   - **Warning**: 最近 AQI > 50 且為室外設施
   - **Safe**: 其他情況

### 情境模擬設計
- **觸發條件**: 所有 AQI 測站值 < 50（空氣品質良好）
- **模擬方法**: 將特定測站（高雄美濃）AQI 強制設為 150
- **驗證目的**: 確保風險標籤邏輯正常運作，產生可見的 High Risk 案例

## 分析結果

### 風險分布統計
| 風險等級 | 數量 | 百分比 | 條件 |
|----------|------|--------|------|
| **High Risk** | 935 | 15.8% | AQI > 100 |
| **Warning** | 378 | 6.4% | AQI > 50 + 室外 |
| **Safe** | 4,612 | 77.8% | 其他 |
| **總計** | 5,925 | 100% | - |

### 情境模擬效果
- **注入測站**: 高雄美濃測站 AQI 從 90 → 150
- **影響範圍**: 142 個避難所受到該測站影響
- **風險提升**: 其中 142 個被標記為 High Risk

### 地理分布特徵
- **High Risk 集中區域**: 主要在高雄美濃測站周邊的屏東縣霧臺鄉
- **Warning 分布**: 散布在 AQI 50-100 範圍測站附近的室外避難所
- **Safe 主體**: 大部分避難所位於空氣品質良好區域

## 技術實現

### 資料處理技術
- **程式語言**: Python 3.x
- **主要套件**: pandas, requests, folium, math
- **API 整合**: EPA 空氣品質監測網即時 API
- **逆地理編碼**: OpenStreetMap Nominatim API

### 視覺化實現
- **地圖框架**: Folium 互動式地圖
- **效能優化**: MarkerCluster 聚合大量標記
- **圖層設計**: AQI 測站、室內避難所、室外避難所分層顯示
- **圖例系統**: 無陰影設計避免瀏覽器效能問題

## 品質保證

### 資料驗證
1. **座標驗證**: 5,859 個避難所通過台灣陸地範圍檢查
2. **邊界檢查**: 使用放寬邊界（±0.2度）和嚴格邊界（±0.1度）雙重驗證
3. **逆地理編碼**: 抽樣驗證 50 個避難所的實際地址

### 錯誤處理
- **API 失敗重試**: 網路請求超時自動重試機制
- **座標異常過濾**: 自動識別並排除經緯度為 0 或超出範圍的記錄
- **資料完整性檢查**: 確保必要欄位不為空值

## 發現與挑戰

### 主要發現
1. **資料品質問題**: 原始資料存在 21 筆明顯錯誤（0.35%）
2. **屬性缺失**: 室內/室外資訊完全缺失，需要人工推斷
3. **座標精度**: 部分避難所座標存在偏移，可能來自 GPS 誤差或轉換錯誤

### 技術挑戰
1. **API 限制**: 逆地理編碼 API 有請求頻率限制
2. **效能優化**: 大量地圖標記造成瀏覽器卡頓
3. **邊界定義**: 台灣各縣市邊界缺乏官方數位資料

## 改進建議

### 短期改進
1. **人工覆核**: 對 66 個座標異常案例進行人工確認
2. **屬性補充**: 聯繫相關部門取得正確的室內/室外分類
3. **邊界精確化**: 使用政府官方行政區邊界 GIS 資料

### 長期發展
1. **氣象整合**: 加入風向、風速、降水等氣象因子
2. **時間維度**: 建立時間序列分析，預測空氣品質趨勢
3. **機器學習**: 開發基於多變數的空氣品質風險預測模型

## 結論

本專案成功建立了台灣避難所空氣品質風險評估框架，從 5,973 筆原始資料中清理出 5,925 筆高品質資料，識別出 935 個高風險避難所。透過系統性的資料清理、空間分析和視覺化，為公共安全決策提供了可靠的資訊基礎。

雖然「最近測站」指標存在簡化限制，但在當前資料條件下提供了實用的風險評估方法。未來可透過整合更多氣象、地形和社會經濟變數，進一步提升分析的精確性和預測能力。
