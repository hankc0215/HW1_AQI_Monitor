# AQI 避難所風險分析反思報告

## 資料完整性

### 原始資料發現的問題

在處理 EPA 避難收容處所資料時，我發現了多個資料完整性問題：

1. **位置錯誤**：18 個避難所存在明顯的座標錯誤，例如「忠孝國小」記錄為彰化市但實際座標不符，「松林社區活動中心」標示為新竹縣新豐鄉但位置錯誤。這些錯誤可能來自資料輸入錯誤或座標轉換問題。

2. **重複項目**：霧峰區存在多個重複的避難所設施，如「霧峰國中行政大樓地下室」重複出現，造成分析結果偏差。

3. **座標異常**：66 個避難所座標超出台灣陸地範圍，其中包含離島地區（如金門、馬祖）的座標錯誤標記。

4. **缺乏室內/室外屬性**：原始資料完全缺乏 `is_indoor` 屬性，這是風險評估的關鍵因素。我透過關鍵字分析（如「地下室」、「禮堂」、「戶外」等）建立分類邏輯，將 4,895 個避難所標記為室內，1,033 個標記為室外，其餘 16 個無法確定。

### 處理缺乏室內/室外屬性的策略

我採用了多層次方法來處理這個困境：
- **關鍵字識別**：分析避難所名稱中的「地下室」、「禮堂」、「戶外」、「廣場」等關鍵字
- **設施類型推斷**：根據「國小」、「國中」、「活動中心」等設施類型推斷
- **人工驗證**：對模糊案例進行人工檢查和標記
- **保留未知類別**：無法確定的案例保留為未知，避免錯誤分類

## AI 協作

### Cascade 表現優異的部分

Cascade 在以下方面表現出色：

1. **資料處理自動化**：快速建立完整的資料處理流程，包括 API 請求、資料清理、座標驗證等
2. **地理計算**：準確實作 Haversine 公式計算兩點間距離，並建立最近鄰居演算法
3. **視覺化整合**：使用 Folium 建立互動式地圖，整合 AQI 測站和避難所位置，並使用 MarkerCluster 優化大量標記的渲染效能
4. **情境模擬設計**：設計情境注入機制，將特定測站 AQI 設為 150 以驗證風險標籤邏輯
5. **問題診斷**：快速識別並處理地圖渲染的效能問題（陰影效果和大量標記）

### 台灣座標系統的「幻覺」問題

在 CRS 方面，Cascade 確實出現了一些「幻覺」：

1. **座標系統混淆**：最初建議使用 TWD97/TM2 但實際上 EPA API 提供的是 WGS84 (EPSG:4326)，需要明確區分
2. **邊界定義錯誤**：第一次實作的縣市邊界過於嚴格，導致大部分避難所被標記為「超出邊界」，後來調整為放寬邊界
3. **座標驗證邏輯**：最初採用單一邊界檢查，後來改為雙層檢查（放寬邊界 + 嚴格遠離檢查）來區分真正的問題案例

這些「幻覺」主要來自對台灣地理資訊系統的不熟悉，但透過迭代調整最終得到了合理的結果。

## 空間推理

### 「最近測站」作為評估指標的合理性

「最近測站」作為評估指標具有以下優點和限制：

**優點**：
- **直觀性**：距離越近，空氣品質越相似，符合一般人的直覺理解
- **實用性**：計算簡單，執行效率高，適合大規模分析
- **基礎性**：提供了風險評估的基本框架，可作為其他複雜模型的基礎

**限制**：
- **忽略地形影響**：山脈、河谷等地形會影響空氣污染物擴散
- **未考慮風向**：下風向的測站可能比上風向更遠的測站更有代表性
- **測站分布不均**：都市地區測站密集，鄉村地區稀疏，可能造成偏差
- **時間延遲**：污染物傳播需要時間，即時距離不能完全代表暴露風險

### 主導開發者應考慮的其他變數

身為主導開發者，應考慮以下變數來提升分析準確性：

1. **氣象因素**：
   - **風向風速**：建立風向玫瑰圖，考慮上風向/下風向測站權重
   - **大氣穩定度**：逆溫層會影響污染物擴散
   - **降水**：降雨可以淨化空氣，降低 AQI

2. **地形因素**：
   - **高程資料**：使用 DEM 數值高程模型分析地形屏障
   - **土地利用**：都市熱島效應、綠地淨化效果
   - **海岸線影響**：海陸風循環對污染物傳播的影響

3. **時間維度**：
   - **時段分析**：上下班尖峰時段 vs 深夜時段
   - **季節變化**：東北季風、西南氣流等季節性影響
   - **歷史趨勢**：使用時間序列分析預測風險

4. **社會經濟因素**：
   - **人口密度**：高密度地區的暴露風險更高
   - **脆弱族群**：老人、兒童、慢性病患者的分布
   - **交通流量**：道路網密度與車流量對空氣品質的影響

5. **技術改進**：
   - **插值演算法**：使用克里金插值等空間統計方法
   - **機器學習**：結合多種變數建立預測模型
   - **即時預警**：建立自動化風險預警系統

## 結論

這次 AQI 避難所風險分析專案展示了資料科學在公共安全應用中的潛力。雖然「最近測站」是一個簡化的評估指標，但透過資料清理、情境模擬和視覺化，我們建立了一個實用的風險評估框架。未來的改進方向應該是整合更多氣象、地形和社會經濟變數，建立更精準的空氣品質風險預測模型。

最重要的是，這個過程提醒我們：資料品質是分析成功的基礎，而 AI 工具雖然強大，但仍需要專業領域知識的指導和驗證。
